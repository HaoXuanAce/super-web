# AI P图网站提示词库大全

> 适用场景：用户上传人物照片，通过勾选功能或输入自然语言提示词，让 AI 自动完成修图。
>
> 建议用法：前端选择功能后，将对应提示词拼接到基础提示词中，再附加统一负向提示词和身份一致性约束。

---

# 一、全局基础提示词

## 1. 通用人像修图基础提示词

```text
请基于用户上传的原图进行高质量人像精修，保留人物真实身份、五官特征、发型、服装、姿势、背景和整体构图，不改变人物是谁。修图效果自然、真实、细腻，避免过度美化和塑料感。保持正常的人体比例、皮肤纹理、面部结构和光影逻辑，最终效果像专业摄影师和商业修图师完成的高级精修照片。
```

## 2. 身份一致性约束

```text
必须保持人物身份一致，保留原人物的脸型特征、眼睛形状、鼻子结构、嘴唇特征、眉毛走势、发际线和整体辨识度。不要生成另一张脸，不要改变年龄层，不要改变性别，不要改变种族特征，不要改变人物身份。
```

## 3. 自然修图约束

```text
所有修改必须自然、克制、真实，不出现明显液化痕迹，不出现边缘扭曲，不破坏背景直线，不改变衣服图案，不改变手指数量，不造成五官错位，不产生蜡像脸、假脸、过度磨皮、锐化过度、颜色溢出或异常光斑。
```

## 4. 高清增强基础提示词

```text
提升照片清晰度、细节、分辨率和质感，修复轻微模糊、噪点、压缩痕迹和失焦问题。增强眼睛、头发、皮肤、服装和背景的真实细节，但不要过度锐化，不要生成虚假的皮肤纹理，不要改变原始内容。
```

## 5. 通用负向提示词

```text
禁止改变人物身份，禁止换脸，禁止生成陌生人，禁止五官错位，禁止脸型严重变形，禁止皮肤塑料感，禁止蜡像感，禁止过度磨皮，禁止背景弯曲，禁止建筑线条变形，禁止肢体扭曲，禁止手指异常，禁止多手多脚，禁止服装图案错乱，禁止文字乱码，禁止严重色偏，禁止曝光过度，禁止锐化过度，禁止不自然阴影，禁止过度液化，禁止假发感，禁止改变人物动作和构图。
```

# 四、皮肤类功能

## 1. 美白皮肤

```text
自然提亮人物面部和身体皮肤，使肤色更加均匀、干净、通透。保留正常血色和皮肤层次，不要变成惨白，不要影响头发、服装和背景颜色。
```

## 2. 局部美白

```text
仅对人物裸露皮肤区域进行自然提亮，包括面部、脖子、手臂和腿部。保持各部位肤色统一，不改变衣服、背景和其他物体的亮度。
```

## 3. 肤色均匀

```text
统一面部、脖子、手臂和身体各区域肤色，修复局部暗沉、色差、泛红和不均匀问题。保持自然真实，不要出现面部和脖子颜色不一致。
```

## 4. 磨皮

```text
轻柔改善皮肤粗糙、毛孔明显和轻微纹理问题，保留适量毛孔和真实皮肤结构。不要形成塑料皮肤，不要把鼻子、嘴唇和眼睛边缘磨糊。
```

## 5. 轻度磨皮

```text
只做轻微磨皮，保留大部分自然纹理，淡化局部粗糙和细小瑕疵，适合日常照片和男性人像。
```

## 6. 中度磨皮

```text
中等强度改善毛孔、粗糙和肤质不均，同时保留关键皮肤纹理和五官边缘，达到自然精修效果。
```

## 7. 去痘痘

```text
去除面部明显痘痘、红肿痘印和局部炎症痕迹，修复区域与周围皮肤自然融合，保留真实皮肤纹理。
```

## 8. 去痘印

```text
淡化面部红色或褐色痘印，均匀肤色，保留自然皮肤质感，不要将整张脸过度磨平。
```

## 9. 去黑头

```text
自然淡化鼻子和面部可见黑头，清洁毛孔观感，保留鼻部结构和皮肤纹理，不要让鼻子失去立体感。
```

## 10. 去毛孔

```text
适度缩小明显毛孔，让皮肤更加细腻平整，同时保留真实细节，避免蜡像感。
```

## 11. 去油光

```text
减少额头、鼻子、脸颊和下巴的明显油光与反光，恢复自然哑光皮肤质感，保留合理高光和立体感。
```

## 12. 增加水光肌

```text
让皮肤呈现自然水润、有弹性、轻微透亮的水光质感。高光要细腻柔和，不要出现油腻、过曝或塑料感。
```

## 13. 去泛红

```text
减少面部局部泛红、鼻翼红、脸颊红和敏感红，同时保留正常血色，不要让肤色灰白。
```

## 14. 去暗沉

```text
改善面部和身体皮肤暗沉发灰的问题，让肤色更明亮、通透、有精神，保持自然色调。
```

## 15. 去黄气

```text
中和肤色中过多的黄色调，让皮肤更干净、自然、通透，但不要偏粉、偏白或失真。
```

## 16. 去斑

```text
自然淡化雀斑、晒斑、色斑和局部色素沉着，修复区域需要与周围皮肤自然融合。
```

## 17. 保留雀斑美化

```text
保留人物原有雀斑特征，仅适度淡化过深或不均匀的部分，让雀斑自然、干净、有辨识度。
```

## 18. 去皱纹

```text
自然淡化额头纹、鱼尾纹、法令纹、嘴角纹和颈纹，保留符合人物年龄的正常皮肤结构，不要让人物突然年轻几十岁。
```

## 19. 淡化细纹

```text
轻微淡化眼周、嘴角和额头细纹，让皮肤状态更好，但保留真实年龄感和表情纹。
```

## 20. 去疤痕

```text
淡化或去除皮肤表面明显疤痕，保持周围肤色和纹理自然连续，不影响人物身份特征。
```

## 21. 去胎记

```text
自然淡化指定区域的胎记或色素区域，保持周围皮肤纹理一致。仅修改目标区域，不影响其他面部特征。
```

## 22. 去晒痕

```text
改善手臂、肩膀、脖子、腿部等位置明显晒痕和肤色分界，让肤色过渡均匀自然。
```

## 23. 身体皮肤精修

```text
对人物手臂、肩膀、背部、腰部和腿部皮肤进行自然精修，改善粗糙、色差、痘印和小瑕疵，保留正常肌肤质感。
```
